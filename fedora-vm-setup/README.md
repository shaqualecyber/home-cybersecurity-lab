# Fedora VM Setup

## Objective
The purpose of this lab was to create a Fedora Linux virtual machine using VMware Fusion on macOS to begin building a personal cybersecurity lab environment.

## Environment
- macOS
- VMware Fusion
- Fedora Linux 44 Workstation Edition

## Skills Practiced
- Virtualization
- Linux installation
- ISO mounting
- VM configuration
- Linux system setup
- Troubleshooting

## Tools Used
- VMware Fusion
- Fedora Linux ISO
- macOS

## Steps Completed

### 1. Created Fedora Virtual Machine
Created a new virtual machine in VMware Fusion and selected the Fedora Linux ISO.

### 2. Configured VM Resources
Assigned RAM, processor cores, and storage resources for the virtual machine.

### 3. Installed Fedora Linux
Completed the Fedora Linux installation process and verified successful boot.

### 4. Configured System Settings
Reviewed system settings including device name, memory allocation, and operating system details.

### 5. Tested Basic Linux Commands
Practiced basic terminal commands and network testing commands such as ping.

## Challenges Encountered
One challenge was learning the differences between VMware Fusion on macOS and VMware Workstation tutorials commonly shown online.

## Results
Successfully created and configured a Fedora Linux virtual machine for future cybersecurity and SOC-focused labs.
